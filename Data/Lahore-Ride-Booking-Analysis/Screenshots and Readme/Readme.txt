# 🚖 Lahore Ride Booking Analysis (SQL + Power BI)

This project analyzes a month's ride data (25,000 rows) from Lahore city using SQL and Power BI.

## 📊 Power BI Dashboard Structure

**5 Pages Total**

1. **Page 1** – Ride Volume & Booking Status  
2. **Page 2** – Top Vehicle Types by Distance  
3. **Page 3** – Customer Ratings & Cancellation Reasons  
4. **Page 4** – Top Customers & Distance Trend  
5. **Page 5** – Driver Ratings & Rating Comparison

---

## 🔍 Insights

- 62% of rides were successful (as expected)
- Top vehicle type: Bike
- Most cancellations: "Driver asked to cancel"
- Prime Sedan had best driver ratings
- Weekend ride volume was highest
